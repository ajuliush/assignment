## Installation

-   `[https://github.com/itminanbiva078/hrms_software.git](https://github.com/ajuliush/assignment.git)`
-   `cd assignment/`
-   `composer install`
-   `cp .env.example .env`
-   Update `.env` and set your database credentials
-   `php artisan key:generate`
-   `npm install`
-   `npm run dev`
-   `php artisan serve`
